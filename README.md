# Responsive Todo List App

A responsive and modern Todo List web application developed using HTML, CSS, and JavaScript. The application helps users organize daily tasks efficiently with features such as task management, filtering, local storage support, and responsive design.

---

# Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Completed)
- Clear completed tasks
- Task counter
- Current date display
- Local storage support
- Responsive user interface
- Modern and clean design

---

# Technologies Used

- HTML5
- CSS3
- JavaScript

Project Description

The Todo List application allows users to manage tasks in a simple and interactive way. Users can add tasks, mark them as completed, remove tasks, and filter tasks based on their status. The application stores data using browser local storage, ensuring tasks remain saved even after refreshing the page.
