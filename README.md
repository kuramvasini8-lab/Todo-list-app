# My Todo List App

A modern and responsive Todo List web application built using HTML, CSS, and JavaScript.

This application helps users manage daily tasks efficiently with features like task filtering, task completion tracking, local storage support, and a clean modern UI.

# Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks
  - All
  - Active
  - Completed
- Clear completed tasks
- Responsive design
- Local storage support
- Task counter
- Current date display
- Smooth animations and hover effects
- 
# Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
